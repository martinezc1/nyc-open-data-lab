
# NYC Open Data Lab

The **NYC Open Data Lab** is a public-facing initiative focused on transforming New York City’s open data into reproducible research, civic insights, and student-driven publications.

Built at the intersection of data science, education, and storytelling, the Lab connects teaching, open-source tools, and public-facing analysis into a single, reproducible ecosystem.

🌐 **Live site:** [<https://martinezc1-nyc-open-data-lab.share.connect.posit.cloud/>]

------------------------------------------------------------------------

## Overview

The NYC Open Data Lab is designed as a connected system where:

-   **Teaching & OER** → introduces reproducible workflows using R and Quarto\
-   **Tools** → provide streamlined access to open data\
-   **Student Research** → produces real-world analyses using NYC Open Data\
-   **Stories** → publish insights as public-facing data narratives\
-   **Projects & Portfolios** → showcase cohesive bodies of work\
-   **Contributors** → highlight the people behind the work\
-   **Talks & Publications** → share and validate the work externally

This structure ensures that work is not temporary, but instead becomes part of a growing, open, and reproducible body of civic data research.

------------------------------------------------------------------------

## Key Components

### 📊 NYC Open Data Stories

A collection of reproducible data analyses exploring New York City using publicly available datasets.

### 🧰 Open Data Tools

A growing ecosystem of R packages that simplify access to open data, including:

-   `nycOpenData`
-   `nysOpenData`
-   `mtaOpenData`
-   `chiOpenData`
-   `laOpenData`
-   `reproresearchR`

### 🎓 Teaching & OER

Courses and open educational resources designed around end-to-end reproducible workflows using R and Quarto.

### 👥 Contributors

Student researchers and contributors who develop and publish work as part of the Lab.

## Tech Stack

-   **Quarto** — website framework\
-   **R** — data analysis and reproducible workflows\
-   **Quarto + R Markdown** — publishing and documentation\
-   **GitHub** — version control and repository management\
-   **Posit Connect** — deployment

------------------------------------------------------------------------

## Philosophy

The NYC Open Data Lab is built on the principle that:

*Relevance drives rigor.*

By working with real data connected to real systems, students engage more deeply and produce higher-quality, more meaningful work.

The Lab is designed so that:

-   work is reproducible
-   outputs are public-facing
-   projects are persistent
-   tools reduce barriers to entry
